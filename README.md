# Simple To-Do List Project

이 프로젝트는 Node.js를 사용한 간단한 웹 서버와 JavaScript로 구현된 할 일 목록(To-Do List) 애플리케이션입니다.

## 주요 기능

- **Node.js 서버**: `http` 모듈을 사용하여 `index.html` 파일을 서빙하는 간단한 서버입니다.
- **할 일 목록 애플리케이션**:
  - 새로운 할 일 추가
  - 완료 상태 토글 (체크박스 및 텍스트 클릭)
  - 할 일 삭제
  - 반응형 디자인 및 깔끔한 UI

## 실행 방법

### 1. 프로젝트 복제
```bash
git clone https://github.com/jwjun-joy/getmini_prj.git
cd getmini_prj
```

### 2. 서버 실행
Node.js가 설치되어 있어야 합니다.
```bash
node server.js
```

### 3. 접속
브라우저에서 다음 주소로 접속합니다:
[http/localhost:3000](http://localhost:3000)

## 기술 스택

- **Backend**: Node.js (Core modules)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
